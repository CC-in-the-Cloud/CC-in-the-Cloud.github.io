Introduction

From the CCDB established Technical Community, the Common Criteria in the Cloud Technical Community (CCitC for short) was formed. The CCitC consists of members of Certification Bodies, labs and vendors working together to create requirements for application security testing.

https://github.com/CC-in-the-Cloud Github Repository
